# Reto Semana 1: Calculadora de Sumas

**Materia:** Programación para Ciencia de Datos (IPN)

## Descripción
Programa en Python que lee un archivo de texto con datos por línea y calcula la suma de cada renglón. Antes de sumar, limpia los datos para eliminar caracteres inválidos y manejar posibles errores en la captura.

## Reglas
- Elimina letras y símbolos que no formen números.
- Si hay decimales, se trunca la parte decimal.
- Si una línea no tiene números válidos, se toma como **0**.

## Uso
Ejecuta el programa desde la terminal indicando el archivo de texto a procesar. El sistema mostrará las sumas por línea o puede guardar los resultados en otro archivo.
