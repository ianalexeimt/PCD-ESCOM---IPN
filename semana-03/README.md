# Reto Semana 3: Analizador de Ventas

**Materia:** Programación para Ciencia de Datos (IPN)

## Descripción
Programa en Python que procesa un archivo de ventas en formato CSV, agrupa las transacciones por producto y calcula métricas consolidadas de rendimiento.

## Reglas
- Agrupa las ventas por producto de forma automática.
- Calcula el total de unidades vendidas, el ingreso total y el precio promedio (formateado a dos decimales).
- Ignora de forma silenciosa las líneas con datos no numéricos, incompletos o con valores negativos.
- El reporte final se ordena de mayor a menor según el ingreso total generado.

## Uso
Ejecuta el programa desde la terminal pasando el archivo CSV mediante la entrada estándar (`stdin`). El sistema imprimirá el reporte consolidado directamente en la salida estándar (`stdout`).
